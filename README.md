# cpu-emulator
This is CPU Emulator.
Code runs simple CPU instruction;
START,
LOAD,
LOADM,
STORE,
JMP,
CJMP,
CMPM,
ADD,
ADDM,
SUB,
SUBM,
MUL,
MULM,
DISP,
HALT.

# For compile code
```bash javac main.java ```
# For run code
```bash java main program.txt ```

Input text is args[0]. That's why you must run code in command line.
